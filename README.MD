## Synopsis

Login Sederhana menggunakan jQuery, Ajax, JSON, PHP, dan MySQL.
CASE tools yang digunakan:
* jQuery 1.11
* bootstrap 3.36
* XAMPP 5.5 (PHP 5.5 dan MySQL 5)
